# 北タイ語におけるラテン文字の字母名。

"A" == ᩋᩮ / เอ
"B" == ᨷᩦ / บี
"C" == ᨪᩦ / ซี
"D" == ᨯᩦ / ดี
"E" == ᩋᩦ / อี
"F" == ᩋᩮ᩠ᨼ / เอฟ
"G" == ᨧᩦ / จี
"H" == ᩋᩮ᩠ᨩ / เอช
"I" == ᩋᩱ / ไอ
"J" == ᨩᩮ / เช
"K" == ᨤᩮ / แฅ
"L" == ᩋᩮ᩠ᩃ 又は ᩋᩮᩖ / เอล
"M" == ᩋᩮ᩠ᨾ 又は ᩋᩮᩜ / เอม
"N" == ᩋᩮ᩠ᨶ / เอน
"O" == ᩋᩰ / โอ
"P" == ᨻᩦ / พี
"Q" == ᨢᩥ᩠ᩅ / ฅิว
"R" == ᩋᩣᩁ᩼ / อาร์
"S" == ᩋᩮ᩠ᩈ / เอส
"T" == ᨴᩦ / ที
"U" == ᩀᩪ / อยู
"V" == ᩅᩦ / วี
"W" == ᨯᩢ᩠ᨷᨷ᩠ᩃᩥᩅ / ดับบลิว
"X" == ᩋᩮᨠ᩠ᩈ / เอกส
"Y" == ᩅᩱ / ไว
"Z" == ᨪᩯ᩠ᨯ / แซด
